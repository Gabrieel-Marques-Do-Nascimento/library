Welcome to Sanny Commands Library.

Sanny Commands Library (SCL) is an attempt to build a complete and consistent class system and commands documentation spanning GTA 3D series games. Heavily based on original naming conventions.

Explore early preview documentation for GTA III and Vice City.

Huge thanks to all researchers and contributors to the GTA modding documentation available on the following websites:

  • https://gtamods.com/wiki/List_of_opcodes
